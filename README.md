# The Node.js App ctrl script

> Warning: Only support unix/linux/macOS now.

## Installation

```sh
$ npm i nodejsctl -g
```

## Usage

```sh
$ # cd into project folder
$ nodejsctl
Usage: nodejsctl {start|stop|status|restart}
$
$ # start app
$ nodejsctl start
$
$ # view app status
$ nodejsctl status
$
$ # stop app
$ nodejsctl stop
$
$ # restart app
$ nodejsctl restart
```
